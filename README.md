# recipe-by-AI-app
In this application you will be able to obtain new recipes made by AI with the ingredients that you choose so that you can enjoy them later.

## Requirements
- iOS 16+
- Xcode 14
- Swift 5

## This project uses:
- UIKit
- VIPER
- Swift package manager
- Dependency Injection
- OpenAISwift
